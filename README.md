# Expense Tracker (Spendly)

A monorepo containing the Expense Tracker apps — a React Native mobile app and a React web app.

## Structure

apps/
├── mobile/ # React Native app (iOS & Android)
└── web/    # React web app

## Apps

### Mobile (React Native)
- Located in `apps/mobile`
- Targets iOS and Android

### Web (React)
- Located in `apps/web`
- React web application

## Getting Started

Each app has its own setup instructions in its respective folder.

## Contributing

- All changes go through pull requests — nothing merges directly to `main`
- One approval required before merging
